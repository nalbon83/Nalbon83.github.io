# GitHub Pages Satirical Website Starter

This is a static website made for GitHub Pages.

## Files
- `index.html` – main page
- `styles.css` – styling
- `script.js` – small navigation highlight script
- `assets/` – your photos

## Publish on GitHub Pages
1. Create a repository on GitHub.
2. Upload all files from this folder.
3. If this is your main personal site, name the repository `yourusername.github.io`.
4. In the repository, go to **Settings > Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/root` folder.
7. Save and wait a few minutes for publishing.

GitHub describes GitHub Pages as a static site hosting service that can publish HTML, CSS, and JavaScript files directly from a repository, and says it is available in public repositories with GitHub Free. [Source](https://docs.github.com/en/pages/getting-started-with-github-pages/what-is-github-pages)

## Easy edits
- Change text directly in `index.html`
- Replace contact details in the contact section
- Swap images in the `assets/` folder
- Adjust colors in `styles.css`

## Tone note
This site is intentionally satirical. If you want a more serious version, rewrite the copy in `index.html` and keep the same layout.
